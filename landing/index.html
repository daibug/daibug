MCP Tools
What your agent can see

Daibug exposes these tools to any MCP-compatible agent.

get_events
Last N events from the unified log, filterable by source and level
get_network_log
All network requests with full request/response detail
snapshot_dom
Serialized DOM tree with computed styles
get_component_state
React component tree with current props and state
capture_storage
localStorage, sessionStorage, and cookies
evaluate_in_browser
Run JavaScript expressions in the page context
replay_interactions
Serialized record of user interactions
clear_events
Clear the event buffer for a fresh debugging session
